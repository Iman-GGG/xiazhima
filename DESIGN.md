# DESIGN.md — 瞎芝麻

## 气质与意象
- 关键词：实验室仪表盘、检察台、规则裁断、无情绪
- 意象锚点：一台老式精密天平 + 老牌交易所电报机的检测台。冷白工作灯下，黑色金属刻度盘上只跳动几个被允许出现的信号灯（绿/红/灰），其余全是哑光的留白与等宽刻度。它不预测，只测量；不诱导，只判定。
- 名字「瞎芝麻」自嘲规则化、非神化——视觉上要呼应这种克制：宁可空，不可花。

## 视觉策略
- 大量负空间 + 网格分隔，所有面板都像贴在工作台上的金属铭牌。
- 关键结论用大字号「卡片标牌」前置呈现，详细数据以等宽刻度 + 表格在下方铺开。
- 不使用渐变、毛玻璃、霓虹光晕；阴影仅用于卡片层级（极轻一层）。
- 信号灯（绿/红/灰）只用在判定徽标与图表标记上，禁止用作大块背景或装饰色。

## 配色方案（Design Tokens）
- 底色：`#F5F4EE` 纸白（旧仪表面板的米白底）
- 文字主色：`#1A1A1A` 墨黑
- 次要文字：`#6B6B66` 砚灰
- 分隔线：`#E2DFD6` 哑米线
- 合规绿：`#2F7D4E`（茶绿，沉稳合规）— **战法判定专用**
- 风险红：`#B83A2A`（朱砂，警示而非血腥）— **战法判定专用**
- 观望灰：`#8C8A82`（砚灰，悬而未决）
- 行情涨红：`var(--quote-up)` —— **仅大势裁定 / 行情涨跌专用**，A 股惯例
- 行情跌绿：`var(--quote-down)` —— **仅大势裁定 / 行情涨跌专用**，A 股惯例
- 强调点（少量）：`#1A1A1A` 墨黑作主按钮底色
- 暗色模式：`#121211` 底 + `#E9E7DF` 字 + 同上信号 / 行情色（饱和度略降）

> 信号色与行情色数值相近但语义独立：信号色描述「规则是否合格」，行情色描述「股价/指数涨跌」。两者不可互用。大势裁定卡中「强势=涨红 / 弱势=跌绿 / 震荡=观望灰」是唯一约定例外，符合 A 股惯例。

## 字体排版
- 中文标题：`Noto Serif SC`（思源宋体）— 金融严肃克制，仅用于页面/卡片大标题。
- 中文正文与 UI：`Noto Sans SC`（思源黑体）— 清晰可读。
- 数字与代码：`JetBrains Mono` — 等宽、价格/百分比/股票代码必用，强化「刻度感」。
- 排版节奏：标题极大且独立成行，正文紧凑，数字独占一行；行高 1.5；段落之间用细分隔线而非大留白堆叠。

## 动效与交互
- 性格：极克制。仅信号状态变化、数字刷新做 120ms 淡入；不要弹跳、不要 scale > 1.02。
- 缓动：默认 `cubic-bezier(0.2, 0, 0, 1)`，进出一致。
- 悬停：表格行底色由 `#F5F4EE` → `#ECEAE0`；按钮仅边框/底色细微切换。
- 滚动：页面级无视差；图表内部 hover 显示十字游标 + 数值卡。

## 页面结构（核心页）
- 顶部窄条 `Header`：左侧站名「瞎芝麻」+ 副标「SF战法 · 规则裁断台」，右侧导航（裁断台 / 个股解析 / 战法学堂）。常驻一行风险免责小字。
- 首页：①大势裁定卡（结论前置：强势/震荡/弱势 + 上证指数 / 活跃市值 / MA5 斜率三项指标）；②今日 B1（表格，结论列在最左）；③今日 B2（规则待补充）；④今日 S1（规则待补充）；⑤样本池速览。
- 个股解析页：①搜索框 + 顶部结论标牌；②四条 B1 条件「合规/不合规」逐项卡片；③信号提醒区；④BBI/KDJ 数值 + K 线指标可视化。
- 战法学堂：长文 + 规则编号锚点；图文「错误案例」用风险红边框框出。

## 设计禁忌
- 不要科技蓝/紫渐变、不要发光卡片、不要 glassmorphism。
- 不要彩虹色图表；K 线/指标图统一墨黑 + 三种信号色，最多再加一档中性灰。
- 不要 emoji 装饰、不要插画式吉祥物；图标只用 lucide 线性图标，描边一致。
- 不要"涨幅红、跌幅绿"反向配色——大势裁定卡严格使用 A 股惯例「红涨绿跌」（`--quote-up` / `--quote-down`）；战法判定的合规/风险继续走信号色（`--signal-pass` / `--signal-risk`）。两套色系不可混用。
- 不要任何荐股、收益、暴富相关文案；通篇克制中性。
